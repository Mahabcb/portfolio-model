# Portfolio

Portfolio est un site internet qui présente des peintures

## Environnement de développement

### Pré-requis

* PHP 7.4
* Composer
* Symfony CLI
* Docker
* Docker-compose
* nodejs et npm

### Lancer l'environnement de développement

composer intall
npm install
npm run build
docker-compose up -d
symfony server:start

## Lancer les tests

php bin/phpunit --testdox
